# C++ Review

A collection of C++ practice problems and exercises covering core language fundamentals.

## Textbook
Problems are sourced from "C++ for Everyone, 2nd Edition" by Cay Horstmann.

## File Naming Convention
Files follow the format `P[Chapter].[Problem].cpp`
For example, `P1.5.cpp` = Chapter 1, Problem 5.

## Topics Covered
- Chapter 1: Introduction
- Chapter 2: Fundamental Data Types
- Chapter 3: Decisions
- Chapter 4: Loops

## Notes
All files are housed in a single Visual Studio project, which only allows one active
`main()` function at a time. As a result, each file's code is commented out by default
to prevent linker conflicts. To run a specific problem, simply uncomment the code in
that file and comment out any others.
Built and tested in Visual Studio on Windows.

## Visual Studio Shortcuts
- Comment selection: `Ctrl+K`, `Ctrl+C`
- Uncomment selection: `Ctrl+K`, `Ctrl+U`
